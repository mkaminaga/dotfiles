dotfiles
====

This is dotfiles for my environment.

Scripts
----
* dotfilelinks.sh
Create symbolic links for dotfiles.
Optimized for UNIX.

* create_link.sh
Create symbolic links for dotfiles.
Optimized for Cygwin/MSYS.

Files
----
* .bash_profile
Login entry script.

* .bashrc
Shell entry script.

* .bashrc.alias
Alias for .bashrc.

* .bashrc.utils
Functions for .bashrc.

* .vimrc
Vim settings.

* .vimrc.keymap
Vim keymap settings.

* .vimrc.plugins
Vim plugin settings.

* .vimrc.util
Vim function settings.

* .vimrc_minimum
Light vim setting.

Doc
----
* howto.txt
Note for machine settings that cannot be automated from script.

Tools
----
* tools/android
Tools for android.

* tools/windows
Tools for windows.
- TERATERM.INI
- VC++_tool_chain_cygwin_launcher.bat
